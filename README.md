# 後で読む - Chrome拡張機能

時間指定で記事を保存し、指定時刻に通知するブックマーク管理拡張機能です。

## 特徴

- **時間指定で保存** - 今夜/週末/来週/カスタム日時を選択
- **デスクトップ通知** - 指定時刻に記事タイトルで通知
- **新しいタブ** - 今日読む記事を一覧表示
- **複数デバイス同期** - Chrome同期機能でデータ共有
- **アーカイブ機能** - 読了記事の管理と一括削除

## インストール

### 開発者モードでのインストール

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/sayasaya8039/bookmark_reader.git
   ```

2. Chromeで `chrome://extensions` を開く

3. 右上の「デベロッパーモード」をON

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

## 使い方

### 記事を保存

**方法1: ツールバーから**
1. 保存したいページでツールバーの📚アイコンをクリック
2. 読みたい時間を選択
3. 「保存する」をクリック

**方法2: 右クリックから**
1. ページ上で右クリック → 「後で読む」
2. サブメニューから時間を選択

### 記事を確認

- 新しいタブを開くと記事一覧が表示されます
- タブで「今日」「すべて」「アーカイブ」を切り替え

### 通知

- 設定した時間に記事タイトルで通知が届きます
- 通知をクリックすると記事が開きます

## ファイル構成

```
bookmark_reader/
├── manifest.json          # 拡張機能設定
├── background/
│   └── service-worker.js  # バックグラウンド処理
├── popup/                 # ポップアップUI
├── newtab/                # 新しいタブUI
├── picker/                # 日時選択ウィンドウ
├── shared/                # 共通モジュール
└── icons/                 # アイコン
```

## 技術仕様

- Manifest Version: V3
- データ保存: chrome.storage.sync
- 通知: chrome.alarms + chrome.notifications
